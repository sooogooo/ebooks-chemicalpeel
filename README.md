## 🚀 本地运行

### 环境要求
- Python 3.7+
- pip

### 快速启动
```bash
# 克隆项目
git clone https://github.com/sooogooo/ebooks-chemicalpeel.git
cd ebooks-chemicalpeel

# 安装依赖
pip install -r requirements.txt

# 启动本地服务
./deploy.sh
# 或者
mkdocs serve
```

访问 http://127.0.0.1:8000 查看电子书

### 构建静态文件
```bash
mkdocs build
```
